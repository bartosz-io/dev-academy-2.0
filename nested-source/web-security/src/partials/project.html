Practical project with real-world features

With this comprehensive training program, you'll learn theoretical knowledge and use it to practically secure a real-life personal money tracker application. This project is composed of five key modules — Auth, Dashboard, Expenses, Settings, and Admin — each designed to challenge and enhance your Web security skills. You'll learn role-based access control (RBAC) to manage different user permissions effectively and implement cutting-edge security mechanisms across both frontend and backend systems. By the end of this program, you will not only understand security concepts but will also have the practical skills to design and safeguard Web applications against sophisticated threats.

Auth module

  • Login, signup, password recovery

Dashboard

  • Read budgets and account summary

Expenses

  • Manage account's expenses

Settings

  • Manage account's users and categories

Admin module

  • Manage active sessions of logged users
